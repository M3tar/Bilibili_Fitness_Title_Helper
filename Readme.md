# B 站健身视频标题生成器

## 简介

这是一个用于自动生成 B 站健身视频标题的浏览器扩展程序。支持通过选择日期、博主、时间段和类型，实时预览符合特定格式的视频标题；用户可以在预览框中手动微调标题，再一键复制到剪贴板，或一键填写到 B 站投稿页。

## 功能特性

* ​**日期选择**​：默认选择今天，支持通过日历选择具体日期，也提供 "今天" 和 "昨天" 的快捷选择按钮。
* ​**快捷选中态**​："今天" 和 "昨天" 按钮会根据当前日期自动高亮，方便确认当前选择。
* ​**博主选择**​：目前支持三位博主 "祖嘉泽"、"陈玉轩" 和 "张峰"，使用按钮形式快速选择。
* ​**时间段选择**​：对于博主 "祖嘉泽"，可选择 "早间" 或 "晚间"（默认晚间）；对于博主 "陈玉轩" 和 "张峰"，固定为 "晚间"。
* ​**类型选择**​：当选择张峰博主且日期为周五时，可选择 "手臂核心"、"有氧双冠" 或 "沉浸有氧"（默认手臂核心）。
* ​**显示开关**​：提供“显示陈玉轩”开关，默认关闭，开启后可显示该博主。
* ​**页面识别**​：在 B 站投稿页点击顶部的“识别当前投稿页”，可从默认视频标题或文件名中提取日期，并根据关键词尝试识别博主。
* ​**识别摘要**​：识别后显示日期、疑似博主，以及当前时间段或训练类型，方便复制前确认。
* ​**实时预览**​：根据用户选择的信息实时生成标题预览，切换选项后自动刷新。
* ​**手动编辑**​：标题预览区域支持直接修改，复制时会复制当前预览框中的内容。
* ​**标题来源提示**​：标题预览旁显示“未识别 / 已识别 / 手动”状态标签，底部操作区同步显示当前填写来源。
* ​**填写结果提醒**​：点击“填写到 B 站”后，结果顶部会用橙色提示本次使用默认日期、识别结果或手动调整内容填写。
* ​**分步填写进度**​：填写到 B 站时会按标题、创作声明、分区、标签和封面逐步显示状态，方便判断当前执行到哪一步；填写完成后可点击 `×` 收起结果。
* ​**标题复制**​：点击“复制标题”按钮，将当前标题复制到剪贴板。
* ​**投稿页填写**​：点击“填写到 B 站”，将当前标题预览写入投稿页，并设置创作声明、分区和标签。
* ​**自动封面**​：点击“填写到 B 站”时，会根据当前博主自动选择插件内置封面并尝试替换投稿封面。
* ​**固定投稿配置**​：默认选择创作声明“内容无需标注”、分区“健身”，并设置标签“减肥”“健身”“训练”。
* ​**标签清理**​：填写标签时会保留默认标签，删除非默认干扰标签，再补齐缺失标签。
* ​**封面弱确认**​：封面流程会模拟打开封面设置、本地上传和裁剪确认；由于 B 站封面弹窗存在异步状态，仍建议提交前人工检查封面预览。
* ​**智能UI**​：界面根据选择自动调整，相关选项动态显示隐藏。
* ​**紧凑布局**​：日期输入、快捷日期按钮和操作区经过压缩排布，“显示陈玉轩”开关放在博主标题行右侧，标题下方增加分割线，减少弹窗滚动并提升层次感。
* ​**吸底操作区**​：底部操作区会吸附在弹窗底部；未滚动到底部时显示阴影，滚到底部完整展示时阴影消失。

## 功能规划 🚀

- [x] 按照固定格式自动生成的B站健身投稿视频标题
- [x] 手动识别当前 B 站投稿页默认标题，提取日期生成标题预览
- [x] 健身视频上传后自动识别，提取日期生成标题 + 按博主自动上传封面，一键完成投稿基础流程

### 文件说明

* `images/`：存放扩展程序的图标文件。
* `covers/`：存放按博主自动选择的投稿封面图。
* `css/`：存放样式文件。

## 安装与使用

### 安装

1. 下载本项目代码到本地。
2. 打开 Chrome/Edge 浏览器，进入扩展程序页面 (chrome://extensions/ 或者  edge://extensions/ )
3. 开启右上角的 “开发者模式”。
4. 点击 “加载已解压的扩展程序”，选择下载的项目文件夹。

### 使用

1. 点击浏览器工具栏中的扩展程序图标，打开弹出页面。
2. 在 “选择日期” 中，通过日历或快捷按钮选择日期；默认日期为今天。
3. 在 “选择博主” 中，选择相应的博主（祖嘉泽、陈玉轩或张峰）。
4. 根据博主选择，需要在 “时间段” 中选择合适的时间段（仅祖嘉泽需要选择，陈玉轩和张峰默认晚间）。
5. 如需从 B 站投稿页默认标题提取日期，先打开对应投稿页，再点击顶部的 “识别当前投稿页”。
6. 在 “标题预览” 中确认生成结果；如需临时调整，可以直接编辑预览框内容。
7. 留意标题预览旁的来源标签和底部状态提示，确认当前内容来自识别、手动调整或默认日期。
8. 点击 “复制标题” 按钮，将当前标题复制到剪贴板；或点击 “填写到 B 站”，将当前标题写入投稿页，并自动设置创作声明、分区、标签和对应博主封面。
9. 留意底部填写进度。基础信息完成后，如果封面弹窗仍然打开，请检查封面预览或手动确认裁剪；查看后可点击结果区右上角 `×` 收起。
10. 如需显示陈玉轩，开启“选择博主”标题右侧的“显示陈玉轩”开关。

## 代码说明

### popup.js

* ​**日历功能**​：通过 `initCalendar` 和 `renderCalendar` 函数实现日历的初始化和渲染，支持月份切换和日期选择。
* ​**事件监听**​：对日期输入框、月份切换按钮、复制按钮、快捷按钮和选项按钮等添加事件监听，实现相应的交互功能。
* ​**快捷日期状态**​：通过 `updateShortcutButtons` 函数同步 "今天" 和 "昨天" 的选中状态。
* ​**页面识别**​：通过 `detectCurrentPageTitle`、`extractDateFromText`、`inferTrainerFromText` 和 `getBestDetectedResult` 函数识别当前 B 站投稿页中的日期和博主关键词。候选文本会优先选择“日期 + 博主关键词”更完整的内容，避免只命中日期但漏掉博主。
* ​**来源状态**​：通过 `setTitleSource` 同步标题来源标签和底部来源提示，区分默认、识别和手动调整状态。
* ​**标题预览**​：根据用户选择的日期、博主、时间段和类型，通过 `getTitle` 和 `refreshPreview` 函数实时更新可编辑标题预览。
* ​**标题复制**​：通过 `copyTitle` 和 `copyToClipboard` 函数将预览框中的当前标题复制到剪贴板。
* ​**投稿页填写**​：通过 `fillCurrentUploadPage` 向投稿页 `content.js` 发送填写请求，将当前标题预览写入 B 站标题输入框，并设置创作声明、分区、标签和博主封面。
* ​**填写进度**​：通过 `FILL_UPLOAD_PROGRESS` 消息在 popup 和 content script 之间同步分步状态，并使用 `runId` 避免旧任务进度串入当前操作；底部操作区会根据滚动位置切换吸附阴影。
* ​**训练类型映射**​：通过 `getTrainingType` 函数实现多博主的训练类型管理，使用对象映射表替代了分散的条件判断，提高了代码可维护性和扩展性。

### manifest.json

* ​**基本信息**​：定义了扩展程序的名称、版本、描述等基本信息。
* ​**权限**​：请求了 `clipboardWrite` 权限用于复制标题，`tabs` 权限用于读取当前标签页信息，并通过 `host_permissions` 与 `content_scripts` 在 B 站投稿页读取候选标题文本。
* ​**封面资源**​：通过 `web_accessible_resources` 允许投稿页脚本读取插件内置封面图。
* ​**图标和弹出页面**​：指定了扩展程序的图标和弹出页面。

## 注意事项

* 本扩展程序支持 Chrome 浏览器。
* 使用“识别当前投稿页”或“填写到 B 站”前，请先打开 B 站投稿页，并在更新扩展后刷新投稿页。
* 自动封面依赖 B 站投稿页当前的封面上传控件结构；如果 B 站页面改版，可能需要手动上传或调整选择逻辑。
* 封面上传属于页面自动化模拟操作，插件能确认已尝试提交封面文件，但 B 站弹窗关闭、裁剪确认和服务器保存状态仍需最终人工检查。
* “填写到 B 站”不会点击 B 站最终发布按钮，提交前仍需人工检查。
